ninolearn.learn.evaluation.evaluation_srmse(model_name, variable_name='mean')[source]¶

Evaluate the model using the standardized root-mean-squarred error (SRMSE) for the full time series. Standardized means that the the the RMSE of each season is divided by the corresponding standard deviation of the ONI in that season (standard deviation has a seasonal cycle). Then, these seasonal SRMSE averaged to get the SRMSE of the full time series..

Parameters

  • model_name (str) – The name of the model.

  • variable_name (str) – The name of the variable which shell be evaluated against the ONI prediction.

Returns

The standardized RMSE for the 0, 3, 6, 9, 12 and 15-month lead time.
